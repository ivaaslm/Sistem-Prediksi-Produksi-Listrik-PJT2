Sistem berbasis Machine Learning untuk memprediksi total produksi energi listrik (kWh) guna mendukung perencanaan operasional PLTA Ir. H. Djuanda.
Dikembangkan menggunakan data historis operasional pembangkit tahun 2020 s.d. 2025.

Tech Stack
1. Language: Python
2. Libraries: Streamlit, Scikit-learn (Random Forest), Joblib, Pandas, Numpy
3. Environment: Google Colaboratory

Fitur Aplikasi
1. Prediksi Harian (1 Hari): Estimasi produksi listrik berdasarkan input tanggal, TMA, dan rencana beban unit (Unit 1–Unit 6).
2. Prediksi Rentang Tanggal: Estimasi berskala (bulanan/jangka panjang) lengkap dengan total akumulasi periode, rata-rata harian, serta opsi unduh laporan rekap format CSV.
